事件（on）

事件绑定


        var bbdChart = new BBDChart();
        bbdChart.init({
            container: 'map',
            type: 'circle',
            config: {
                nodeSize: 12
            }
        });

       bbdChart.on('nodeClick', function (data, e) {
            // data：触发事件的对象  e：事件对象
        });

可绑定的事件名: nodeClick, nodeRightClick, edgeClick, edgeRightClick, stageRightClick, stageClick, mouseover, mouseout
